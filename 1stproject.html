What is Pair Programming?

Pair Programming is a team technique for software development. Working in a team of two programmers but only one work station/project. First programmer goes as a "driver" and the second programmer as a "observer/navigater". The driver does the coding while the observer watch over it all and comes with new ideas and inputs. These roles are swapped frequently. Successful Pair programming relies on communication as much as programming skill to be sucsessful.
